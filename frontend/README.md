# Drug Tracker Frontend

Este projeto foi gerado manualmente devido à ausência do Angular CLI no ambiente de criação.

## Como preparar o ambiente

Certifique-se de ter o Node.js instalado.

1. Abra um terminal nesta pasta `frontend`.
2. Instale as dependências:
   ```bash
   npm install
   ```

## Servidor de Desenvolvimento

Execute `npm start` (ou `ng serve`) para iniciar o servidor. Navegue para `http://localhost:4200/`.

## Build

Execute `npm run build` (ou `ng build`) para gerar os arquivos de produção. Os arquivos serão gerados na pasta `dist/`.
